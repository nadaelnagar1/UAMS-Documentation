    Introduction

    UAMS Clinic Block Request is an online system that support the Provider Time Unavailable policy, To ensure all clinical schedule appropriately designate provider unavailability and that “unavailable time” is communicated in a timeframe that minimizes both disruption to clinical operations as well as negative impact on patient care.

    It allows user to enter requests and allows administrators to track and report on requests for clinic blocks and operating room (OR) releases

    Project Structure

    Admin

    The Admin Page allows you to access different sections based on your role within the system, including Systems, Clinic, and People.

    Systems

    The Systems section is where authorized users can access and perform operations on Permission Definitions, Roles, and Active Directory Management.

    Permission-Definition

    Permission Definition is the section where authorized users can view the permissions for every page within the system.

    Roles

    Roles is the section where authorized users can :

    • View all existing roles

    • Create new roles

    • Assign permissions to roles

    • Update existing roles along with their associated permissions.

    ActiveDirectory Management

    Active Directory Management is the section where authorized users can configure LDAP authentication settings, as well as save and reset them as needed.

    Help Configuration

    Help Configuration is the section where authorized users can configure email settings and save/edit it.

    Clinic Management

    Clinics

    Add or edit or delete or view clinic and assign users to clinic or remove users from clinic:

    • View all existing clinics

    • Create new clinic

    • Edit clinic

    • Assign users to clinic

    Request Reason

    Add or edit or delete or disable reason for request:

    • View all existing request reasons

    • Create new reason

    • Edit clinic

    • Assign users to clinic

    People

    The People section is where authorized users can access and perform operations on Users and Providers.

    Users

    Users is the section where authorized users can :

    View a list of all users and the clinics they are assigned to. It is possible to filter the list based on clinic and user status.

    Add new users

    Edit existing users

    Impersonate other users if they have been granted delegate access

    Disable users

    Delete users.

    Providers

    Providers is the section where authorized users can :

    view a list of providers and search for specific providers by name, edit provider information, delete providers from the system, and perform other relevant actions related to provider management.

    Request

    Request is the section where authorized users can view a list of requests and apply filters to narrow down the results.add new requests, edit existing ones, and delete requests from the system.

    Report

    The Report is the section where authorized users can generate an Excel sheet of filtered requests. This feature enables users to generate reports that meet specific criteria and export the results as a downloadable Excel file.

    Calendar

    The Calendar s the section where authorized users can view requests represented in a calendar format and apply filters to narrow down the results. This feature enables users to visualize requests and their corresponding dates, making it easier to manage and track requests over time.
